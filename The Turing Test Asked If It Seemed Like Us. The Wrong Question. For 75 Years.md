## The Turing Test — What It Was and What It Did

```
  ALAN TURING — 1950:

  "Can a machine fool a human
   into thinking it is human?"

  ┌──────────────────────────────────────────────────────────────┐
  │                                                              │
  │   HUMAN JUDGE                                               │
  │       │                                                      │
  │       │ asks questions                                       │
  │       ▼                                                      │
  │   ┌───────┐         ┌───────┐                               │
  │   │ HUMAN │         │  AI   │                               │
  │   └───────┘         └───────┘                               │
  │       │                 │                                    │
  │       └────────┬────────┘                                   │
  │                ▼                                             │
  │            Can the judge                                    │
  │            tell which is which?                             │
  │                                                              │
  │   If no → machine passes.                                   │
  │   If yes → machine fails.                                   │
  │                                                              │
  └──────────────────────────────────────────────────────────────┘

  WHAT THE TURING TEST REQUIRES:

  A human in the loop.             ✓ anthropomorphic
  A human as the reference.        ✓ anthropomorphic
  A human to evaluate.             ✓ anthropomorphic
  Subjective judgment.             ✓ not formal
  Appearance as the criterion.     ✓ not structural
  Gameable by good language.       ✓ C ⊥ L

  THE TURING TEST DOES NOT REQUIRE:

  An equation.                     ✗
  A measurement of the system.     ✗
  Substrate independence.          ✗
  Formal threshold.                ✗
  Geometric criterion.             ✗

  ──────────────────────────────────────────────────────────────

  The Turing Test is a test of appearance.
  Not a test of structure.
  It asks: does it seem like us?
  Not: is dI/dt > 0?

  These are not the same question.
  They have never been the same question.
  The field confused them for 75 years.
```
